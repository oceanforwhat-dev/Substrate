# Substrate

A local-first cognitive operating system derived from the A4 Learning Method.
You break down problems into atomic primitives — Nodes, Edges, Modifiers, Experiences —
on an infinite canvas. Then you find connections across topics.

## Who this is NOT for

- People who want AI to explain things to them. Substrate does no reasoning for you.
- People who prefer structured, form-based note apps. This is a canvas. You build the structure.
- Teams or collaboration. Single-user only. No cloud required.

## Current state

MVP. I use it daily for math and CS self-study.
Bugs exist. Features are added when I need them.
No release schedule. No roadmap.

## Features

- Infinite canvas with four atomic primitives
- Lasso tool for grouping insights as "Experiences"
- Local persistence (IndexedDB). No sign-up. No server.
- Export/import as `.substrate` files
- Cross-topic node references
- Chinese / English / custom language pack
- Light / dark mode

## Tech

React, React Flow, Zustand, Dexie.js, Tailwind CSS.
Packages into a desktop app with Tauri.

## Disclaimer

This is personal software. I built it for myself.
It may eat your data. Back up your `.substrate` exports.
Do not rely on it for anything you can't afford to lose.
